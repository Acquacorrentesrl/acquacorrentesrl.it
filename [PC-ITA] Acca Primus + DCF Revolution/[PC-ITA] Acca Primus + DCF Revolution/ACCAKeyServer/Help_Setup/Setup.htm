Installazione Multi User con Unica Chiave di Rete

 

Per installazioni Multi User con unico numero di serie del programma è previsto l’utilizzo di un’unica chiave di protezione hardware di rete.

In questo caso, per procedere all’installazione, è necessario effettuare le seguenti operazioni:

1) Installare il programma ACCAKeyServer per la gestione della chiave di rete sul computer adibito a SERVER della Chiave effettuando le seguenti operazioni:

 

ATTENZIONE! Il computer SERVER della Chiave di Rete può essere un qualsiasi computer della rete su cui risulti installato Windows 98/Me oppure Windows 2000/XP/2003.

Prima di procedere all’installazione è necessario accertarsi:

  • che ogni altra applicazione in uso sia chiusa;

  • di aver avuto accesso al sistema con i privilegi di Amministratore in caso di installazione su Windows 2000/XP.

 

  • nella finestra Scegliere il prodotto da installare selezionare l’opzione Installa ACCAKeyServer per avviare la procedura di installazione della chiave di rete;

  • procedere seguendo le istruzioni fornite dal programma di installazione.

2) Connettere la chiave hardware ad una porta USB del computer SERVER della Chiave di Rete e procedere all’installazione dei suoi driver secondo le modalità riportate nel capitolo 2 del manuale d’uso.

3) Installare il programma (v. capitolo 2 del manuale d’uso) su tutti i computer CLIENT e sul computer SERVER della Chiave di Rete, se questo sarà utilizzato come postazione di lavoro del programma.

4) Per ogni computer su cui è stato installato il programma effettuare le seguenti operazioni:

  • avviare il programma; al primo avvio un messaggio segnala che il SERVER della chiave di protezione non è stato definito;

  • pigiare il bottone OK del messaggio per chiuderlo; questa operazione attiva il dialog "Configurazione (chiave di protezione)"

  • nel dialog Configurazione (chiave di protezione) specificare il nome del computer SERVER della Chiave;

  • avviare nuovamente il programma per verificare se il SERVER della Chiave è stato specificato correttamente.
